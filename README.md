# Realtek Ethernet Optimizer

A PowerShell script to maximize performance and minimize latency for Realtek PCIe GbE Family Controllers.

## Features
- ✅ **Auto-detects** Realtek Network Adapters.
- ✅ **Checks for Admin Privileges** before execution.
- ✅ **Disables Power Saving** features (Green Ethernet, Energy Efficient Ethernet, etc.).
- ✅ **Reduces Latency** by optimizing Interrupt Moderation and Flow Control.
- ✅ **Detailed Logging** showing "Before" and "After" configurations.

## How to Use
1. Download `ethernet.ps1`.
2. Right-click and select **Run with PowerShell** (ensure you have Admin rights).
3. Follow the on-screen prompts.
